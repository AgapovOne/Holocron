# Holocron
A Star Wars database with list of characters.

### API:  
[Star Wars API - swapi.co](https://swapi.co)

### Features:
- Character list
- Search by name
- Character details
- Local storage of last characters
- Loading indicators
- Error handling

### Libraries:
- RxSwift
- Alamofire
- Kingfisher
- Moya
- DefaultsKit (Codable support for UserDefaults)
